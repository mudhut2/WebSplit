About Timer Splitter

Current Functionality

  • Start, stop, and split timer functionality.
  • amount of splits to stop at opition
  • Page navigation to explore various features.
  • download splits as json to upload and compare(latter not implemented yet)

Technologies Used

This project makes use of modern web technologies:

  • Bootstrap for responsive design and layout.
  • HTML for the structure of the page.
  • CSS for styling and custom layout tweaks.
  • JavaScript for implementing dynamic timer functionality.

Future Functionality

  • upload split view and compare functionality
  • make whiscash react to good or bad splits
  • Allow users to set customizable timer intervals.
  • Save and load timer data from local storage.
  • Improve the visual interface for split times.
  • Enable a dark mode for better accessibility.
